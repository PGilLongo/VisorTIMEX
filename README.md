## Descripción
TIMEX es una aplicación web progresiva (PWA) en desarrollo, diseñada para su futura aplicación en servicios de emergencias. El sistema permite la gestión y monitorización de recursos de forma multiplataforma, accesible desde distintos dispositivos.

## Aplicación TIMEX (PWA)
https://pgillongo.github.io/TIMEXOpBETA/

La aplicación puede instalarse como una app nativa en dispositivos compatibles:
- Android (Chrome)
- iOS (Safari)
- Escritorio

### Instalación
- **Android:**  
  Menú ⋮ → Instalar aplicación / Añadir a pantalla de inicio
- **iOS:**  
  Botón Compartir → Añadir a pantalla de inicio

## VISOR TIMEX (Navegador)
https://pgillongo.github.io/VisorTIMEX/

Aplicación web destinada a la monitorización de los recursos que tienen TIMEX activo. No requiere instalación y se ejecuta directamente desde el navegador.

## Estado del proyecto
Aplicación en fase de desarrollo. Pensada para su implementación y validación en entornos operativos de emergencias.
